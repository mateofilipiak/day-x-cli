# Day X CLI

Simple habit tracking CLI application written in Python.

## Features
- Start new day
- Track progress
- Profile system
- JSON data persistence
- Anti-cheat mechanism (prevents fake day increments)

## Tech stack
- Python 3
- JSON storage
- Git

## Purpose
Learning backend fundamentals: application logic, state handling and version control.

## How to run

Run the application from the project folder:

python main.py
