# Day X CLI

Simple habit tracking CLI application written in Python.
The application allows users to manage daily progress, track habits, and maintain personal data with input validation and persistent storage.

## Features
- Start new day
- Track progress
- Profile system
- JSON data persistence
- Anti-cheat mechanism (prevents fake day increments)

## Tech stack
- Python 3
- JSON storage
- Git

## Purpose
This project demonstrates backend fundamentals such as application logic, state management, input validation, and data persistence using JSON.

## How to run

Run the application from the project folder:

python main.py

## Example usage

1. Start the application
2. Choose an option from the menu:
   - Start day
   - Add habits
   - Track progress
3. Enter data when prompted (e.g. weight, height, reflections)
4. End the day to save progress

The application stores data locally in JSON format.
